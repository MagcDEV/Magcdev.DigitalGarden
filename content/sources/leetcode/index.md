# LeetCode Sources

Raw problem solutions and notes. The LLM compiles these into structured wiki entries under [[wiki/dsa/index|DSA Wiki]] and [[wiki/patterns/index|Patterns Wiki]].

When adding a new problem, use this format:

```markdown
---
title: "LC-XXXX: Problem Name"
source_type: leetcode
source_url: https://leetcode.com/problems/problem-slug/
difficulty: easy | medium | hard
tags:
  - source
  - leetcode
  - pattern-name
date: YYYY-MM-DD
processed: false
---
