Is the ability of an object, variable or method to take multiple forms for example a single interface can be uses to refer to actions or objects of different types and the specific action performed will depend  on the actual type of the object at runtime.

#### Method Overloading:
Allows you to have multiple methods in the same class with the same name but with different parameter lists.

#### Method Overriding:
A subclass provides a specific implementation for a method that is already defined in its superclass, in csharp requires that the base class method to be marked as virtual or abstract (or be part of an interface implementation).
